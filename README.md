# 🧳 Travel List

A simple and intuitive React application to help you organize and track your travel packing list. Never forget essential items for your trips again!

## ✨ Features

- **Add Items**: Easily add items with customizable quantities (1-20)
- **Track Progress**: Mark items as packed with a simple checkbox
- **Smart Sorting**: Sort your list by:
  - Input order (default)
  - Alphabetical description
  - Packed status
- **Delete Items**: Remove individual items or clear the entire list
- **Real-time Statistics**: Track your packing progress with live percentage updates
- **Completion Celebration**: Get a special message when you've packed everything!

## 🚀 Getting Started

### Prerequisites

- Node.js (version 14 or higher recommended)
- npm (comes with Node.js)

### Installation

1. Clone the repository or download the project files

2. Navigate to the project directory:
```bash
cd travel-list-main
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm start
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📖 How to Use

1. **Adding Items**:
   - Select the quantity from the dropdown (1-20)
   - Type the item description in the text field
   - Click "Add" or press Enter

2. **Marking as Packed**:
   - Click the checkbox next to any item to mark it as packed
   - Packed items will appear with a strikethrough

3. **Deleting Items**:
   - Click the ❌ button next to an item to remove it
   - Use "Clear list" to remove all items (confirmation required)

4. **Sorting**:
   - Use the dropdown menu to sort by:
     - Input order
     - Description (alphabetically)
     - Packed status

5. **Track Progress**:
   - View your packing statistics at the bottom
   - See the percentage of items you've packed
   - Get a completion message when you reach 100%!

## 🎯 Available Scripts

### `npm start`
Runs the app in development mode at [http://localhost:3000](http://localhost:3000). The page will reload when you make changes.

### `npm test`
Launches the test runner in interactive watch mode.

### `npm run build`
Builds the app for production to the `build` folder. It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run eject`
**Note: this is a one-way operation. Once you `eject`, you can't go back!**

If you need full control over the build configuration, you can `eject` at any time. This will copy all configuration files and dependencies into your project.
